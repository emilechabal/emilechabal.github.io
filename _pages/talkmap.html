I first built this website at the height of the pandemic, during which many of us were confined at home. This made me nostalgic about academic and non-academic travel, so I used some tools on this website to create a dynamic map of all the places I had given talks since 2015. I have not updated the map since 2022, but I have kept it here as a record of all the lovely friends and colleagues who made this travel possible!

PS. Recordings of some of these talks can be found in the press and media section.
